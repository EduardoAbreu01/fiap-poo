# Projeto FiapRide
O FiapRide é um sistema construído em Java para simular funções de um aplicativo de transporte. 
Atualmente, o projeto foca em demonstrar o funcionamento de uma **Câmera** de monitoramento.
A classe `Camera` simula uma gravação em tempo real que reduz a capacidade do HD gradativamente.
Após o término da filmagem, o console permite ao usuário optar por limpar a nuvem/HD e restaurar o espaço.
Para testar a simulação, basta compilar e executar a classe `SistemaPrincipal.java` na sua IDE.
